# Built Simple Research for Obsidian

Search PubMed, ArXiv, and Wikipedia directly from your Obsidian notes.

## Features

- **Multi-source search**: Query PubMed, ArXiv, and Wikipedia simultaneously
- **Quick citation insert**: One-click citation insertion into your notes
- **Selection search**: Select text and search for related research
- **Customizable**: Enable/disable sources, adjust result limits

## Commands

- `Built Simple: Search selected text` - Search the currently selected text
- `Built Simple: Search PubMed` - Open PubMed search
- `Built Simple: Search ArXiv` - Open ArXiv search  
- `Built Simple: Search Wikipedia` - Open Wikipedia search

## Installation

### From Community Plugins (Coming Soon)
1. Open Obsidian Settings
2. Go to Community Plugins
3. Search for "Built Simple Research"
4. Install and enable

### Manual Installation
1. Download the latest release
2. Extract to your vault's `.obsidian/plugins/obsidian-builtsimple/` folder
3. Enable the plugin in Settings > Community Plugins
